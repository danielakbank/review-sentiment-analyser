# 🛍️ Amazon Review Sentiment Analyser

Binary sentiment classification of Amazon product reviews using NLP and machine learning.

## 🔗 Live Demo
[Click here to try the app](YOUR_STREAMLIT_URL)

## 📌 Overview
- Dataset: Amazon Fine Food Reviews (500k+ reviews)
- Baseline: TF-IDF + Logistic Regression — Accuracy ~89%
- Advanced: Feedforward Neural Network — Accuracy ~86%

## 🛠️ Tech Stack
Python, Scikit-learn, TensorFlow, Streamlit, Pandas, Matplotlib

## 🚀 Run Locally
​```bash
git clone https://github.com/YOURUSERNAME/amazon-sentiment-analysis.git
cd amazon-sentiment-analysis
pip install -r requirements.txt

​```
